Всесезонна трансмісійна олива XADO 80W-90 GL-3/4/5 RED BOOST

Ціна: 450 грн

Об’єм: 1 л

XADO 80W-90 GL-3/4/5 RED BOOST – надійна універсальна трансмісійна олива для легкових, вантажних автомобілів та спеціальної техніки. Включає REVITALIZANT та інноваційну присадку RED BOOST для захисту від зносу та тривалого ресурсу агрегатів.

Рекомендована для механічних коробок передач, диференціалів, роздавальних коробок та гіпоїдних передач, де вказано API GL-3/4/5. Забезпечує стабільну роботу під високими навантаженнями та температурами.

Переваги продукту

  • Висока стабільність властивостей при інтенсивній експлуатації
  • Стійка до старіння та окислення, навіть при пікових температурах
  • Зменшує шум і вібрації під час роботи трансмісії
  • Особливо ефективна для гіпоїдних та високонавантажених передач
  • Відмінний протизносний та антикорозійний захист, мінімальне піноутворення
  • Універсальна олива для різних типів автомобілів та техніки
  • REVITALIZANT компенсує часткове зношення деталей тертя
  • RED BOOST підтримує оптимальний рівень лужності навіть при максимальних навантаженнях
