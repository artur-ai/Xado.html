Олива для двигуна XADO Atomic Oil 10W-40 4T MA2 RED BOOST

Ціна: 470 грн

Об’єм: 1 л

Напівсинтетична моторна олива для 4-тактних двигунів мотоциклів — XADO Atomic Oil 10W-40 4T MA2 RED BOOST. Формула містить РЕВІТАЛІЗАНТ® RF[10,5]* і сучасний комплекс RED BOOST для стабільної роботи двигуна.

(Підходить і для використання в генераторах будь-якого типу!)

Ця олива створена для стабільного змащування чотиритактних моторів у мотоциклах та генераторах. Основа з синтетичних ефірів у поєднанні з пакетом присадок забезпечує надійний захист деталей від зношування, утворення відкладень і перегріву.

Рекомендована для трансмісій з мокрим зчепленням у туристичних, спортивних та кросових мотоциклах, де потрібні масла класу API SF, SG, SH, SJ, SL чи SM.

Основні переваги

  • Швидкий запуск двигуна навіть у холодну пору та надійне змащування в складних умовах
  • Мінімальний рівень випаровування зменшує витрати оливи на чад
  • Зниження тертя — вищі експлуатаційні характеристики та економічність
  • Безшумне та плавне перемикання передач, стабільна робота мокрого зчеплення
  • Висока миюча здатність підтримує чистоту деталей двигуна
  • Активний захист від зношування завдяки дії РЕВІТАЛІЗАНТ®
  • RED BOOST контролює рівень лужності під час високих теплових навантажень

Відповідність стандартам

Продукт відповідає вимогам міжнародних специфікацій:

  • API SN
  • JASO MA/MA2

Сфера застосування

Призначена для чотиритактних моторів мотоциклів, скутерів, квадроциклів та іншої мототехніки. Забезпечує надійність навіть у навантажених режимах.

Також підходить для генераторів, гарантуючи їх безвідмовну роботу та захист двигуна впродовж тривалого часу.
